## 인증 UX 확정 사항

- 재설정 링크 = 1회용. 링크 세션은 비밀번호 저장 전까지 다른 페이지 차단, "취소하고 기존 비밀번호 유지" 탈출구 제공
- 사용된 링크 재클릭: 상태별 분기 (잠금 중→변경 화면+안내 / 완료 후→대시보드 / 비로그인→로그인+만료 안내)
- `/reset/update`는 재설정 링크 경유로만 접근 가능 (서버 액션도 동일 검사)
- 입력 필드: X 지우기 + 비밀번호 눈 토글 (`components/ui/TextField.tsx` — 신규 폼 필수 사용)
- 로그인 상태 유지: 체크박스 (기본 해제 = 브라우저 종료 시 로그아웃)
